Candy Sales Data Analysis Dashboard

A Python-based data analysis project that explores candy sales data using Pandas, Matplotlib, and Seaborn. This project analyzes sales trends, top-selling products, regional performance, and monthly patterns through data visualization and statistical insights.

Features
Calculate total and average sales
Analyze yearly sales trends
Identify top-selling candy products
Visualize sales distribution
Compare regional sales performance
Examine monthly sales patterns
Generate correlation heatmap
Create pairplot for numerical analysis
Technologies Used
Python
Pandas
Matplotlib
Seaborn
Project Objective

The objective of this project is to analyze candy sales data and extract useful business insights through data processing and visualization techniques.

Output

This project provides meaningful insights into sales performance, product demand, customer trends, and regional growth using simple and effective visual analysis.
